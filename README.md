# 三心譯記 TriMind

> 即時翻譯與會議記錄，一個網頁搞定。
> 出國開會聽不懂時當翻譯機，平常開會當記錄器，還能用 AI 自動整理會議重點。

這是一個純前端的網頁小工具，打開瀏覽器就能用，也可以「加到主畫面」像 App 一樣使用。免費分享給大家，歡迎自由使用。

---

## 功能特色

### 兩種模式，一鍵切換

- **即時翻譯**：邊聽邊翻，支援英、日、韓、德、法、西、泰、越、中等語言，也能「自動偵測」對方說的語言。每句話留下時間、語向與譯文。
- **會議記錄**：純語音轉文字，不翻譯，適合中文會議做逐字稿。可在過程中插入「重點標記」分段。

### AI 加值（選用）

- 設定個人的 Gemini API key 後，翻譯由 AI 處理，品質更好、能理解專業術語。
- 可填入「會議主題」提示，讓 AI 更準確處理領域用語與縮寫。
- **AI 會議摘要**：一鍵把整場逐字稿整理成「重點 / 決議 / 待辦事項」。
- 未設定 key 時，自動使用免費翻譯服務，功能不中斷。

### 紀錄與匯出

- 即時逐字稿，每句帶時間戳。
- 匯出排版精美的會議報告（可列印成 PDF），或純文字逐字稿。
- 自動存檔，意外關閉或當機可復原。
- 錄音時螢幕保持喚醒，長時間聆聽不中斷。

## 開始使用

1. 用手機或電腦的瀏覽器開啟工具網址（建議使用 **Chrome**，語音辨識支援最完整）。
2. 第一次使用時，允許麥克風權限。
3. 選好語言，按下麥克風即可開始。
4. 手機可透過瀏覽器選單「加到主畫面」，之後像 App 一樣開啟。

### 啟用 AI 翻譯（選用）

1. 到 [Google AI Studio](https://aistudio.google.com) 登入 Google 帳號。
2. 點「Get API key」→「Create API key」，取得一組 `AIza` 開頭的金鑰。
3. 在工具裡點「⚙ API」，貼上金鑰並儲存。

> Gemini 提供免費額度，個人使用通常足夠，且不需綁定信用卡。

## 使用建議

- 語音辨識在 **Chrome（尤其 Android）** 效果最好。
- 收音以「真人正常說話、靠近裝置」效果最佳；背景音樂、多人同時交談、遠距離會影響辨識準度。
- 需要網路連線（語音辨識與翻譯都需連網）。
- 若要收電腦喇叭或線路的聲音，可在「⚙ API」面板開啟「關閉回聲消除」選項試試（效果視裝置而定）。

## 隱私

所有設定與紀錄（API key、會議主題、逐字稿）**只儲存在你自己裝置的瀏覽器中，不會上傳到任何伺服器**。翻譯內容僅會傳送給你所選擇的翻譯服務（Google 翻譯或你自己的 Gemini API）。
