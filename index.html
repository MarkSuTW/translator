Gemini API Key 設定

貼上 Gemini API key（AIza 開頭），啟用 AI 翻譯（品質較佳、懂專業術語並可自動偵測語言）。
Key 只存在這支手機的瀏覽器，不會上傳伺服器。未設定時自動使用 Google 翻譯。

取得方式：aistudio.google.com → Get API key → Create API key

收音進階設定

關於「叮咚」聲：那是手機每次「重新聆聽」的系統提示音。收不到聲音時它會不斷重試、不斷叮咚。打開上面的開關讓它收得到聲音後，重試會減少、叮咚也會跟著大幅變少。那聲音本身屬手機系統層級，網頁無法直接靜音，但多數手機可在系統設定關閉語音輸入提示音。
